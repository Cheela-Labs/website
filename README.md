# Cheela Website

The official website for **Cheela**.

This repository contains the source code for the public-facing website, including the landing page, product information, documentation links, blog, careers, and other company content.

> Building the infrastructure for reliable AI agents.

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion
- Biome
- pnpm

---

## Getting Started

### Prerequisites

- Node.js 22+
- pnpm

### Installation

```bash
git clone https://github.com/Cheela-Labs/website.git
cd website

pnpm install
```

### Development

```bash
pnpm dev
```

Open http://localhost:3000 to view the application.

### Production Build

```bash
pnpm build
pnpm start
```
